[탭 구성]
  1. 지급 대기 목록
  2. 이체 완료 내역
  3. 세금계산서 (포인트 차감 내역)

[급여 주기 유형]
  - 일급 (daily)
  - 주급 (weekly)
  - 월급 (monthly)

[세금계산서 조회]
  조회 기간: 지난주 / 지난달 / 이번달 / 직접 선택
  출력 항목: 유저명, 상호, 사업자번호, 이메일, 공급가액, 세액(10%), 합계

[API 엔드포인트]
  GET  /api/payroll/pending       → 대기 중인 급여 목록
  GET  /api/payroll/paid          → 이체 완료 내역
  GET  /api/payroll/stats         → 통계
  POST /api/payroll/generate      → 급여 생성
  POST /api/payroll/export        → 엑셀 출력 (이체 완료 처리)
  GET  /api/payroll/{id}/pdf      → PDF 다운로드
  GET  /api/invoice/weekly-summary → 세금계산서 요약
  GET  /api/invoice/export-tax    → 세금계산서 엑셀 다운로드
  POST /api/payroll/create-test-data → 테스트 데이터 생성
